# 🔐 Login & Sign-Up Page

A responsive authentication UI with real-time form validation and session persistence using Local Storage.

## 🔗 Live Demo
[View Live](https://github.com/haneen562/login-signup-page-)

## ✨ Features
- Real-time client-side form validation (email format, password strength)
- Local Storage integration to persist session data across page reloads
- Smooth toggle between Login and Sign-Up forms
- Fully responsive on all screen sizes

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6+)
- Local Storage API


## 🚀 How to Run
1. Clone the repo: `git clone https://github.com/haneen562/login-signup-page-`
2. Open `index.html` in your browser
